4 ⚠️ Cuidado — Rapariga

🚨 Palavra ofensiva no Brasil

Em espanhol, "rapaza" ou "muchacha" é completamente normal.
Em Portugal, "rapariga" significa "moça / garota" — sem problema.

No Brasil, RAPARIGA é uma palavra grosseira e ofensiva.
Equivale a "perra / zorra" — nunca use no Brasil!

✅ Use no Brasil: moça · garota · menina · mulher

💡 Regra de ouro:

O mesmo português tem palavras diferentes em Portugal e no Brasil.
Sempre aprenda o português do Brasil neste curso!

5 😎 CARA — 3 Significados

💡 Contexto é tudo!

A mesma palavra CARA pode ter 3 significados completamente diferentes. O contexto diz tudo.

😊
Cara
(KÁ-ra)
Cara / Rostro
"Ela tem uma cara bonita."
💸
Cara / Caro
(KÁ-ra / KÁ-ru)
Caro/a / Costoso/a
"Esse celular é muito caro."
🧑
Cara
(KÁ-ra)
Man / Tipo / Parce / Pana
"Esse cara é meu amigo."
🗣️ Exemplos em contexto — clique para ouvir:

▶ Ela tem uma cara bonita. — Ella tiene una cara bonita. (rostro)
▶ Esse restaurante é caro pra caramba! — ¡Ese restaurante es carísimo! (precio)
▶ Aquele cara trabalha comigo. — Ese tipo/man trabaja conmigo. (persona)

6 ✏️ Exercícios

1. Como se chama uma mulher com mais de 60 anos, de forma respeitosa?
✅ Senhora ou Idosa — também pode usar Dona + nome (ex: Dona Maria)
2. Qual a diferença entre Menino e Garoto?
✅ São sinônimos! Os dois significam "Niño" (3-12 anos). Garoto/Garota é mais informal e moderno no Brasil.
3. Complete: "Aquele ___ tá chegando agora." (homem jovem, informal)
✅ cara / rapaz / moço — todas certas! "Aquele cara tá chegando agora."
4. Por que NUNCA usar "rapariga" no Brasil?
✅ No Brasil, rapariga é uma palavra ofensiva e grosseira. Use: moça, garota, menina.
5. "Esse cara é caro." — quantos significados de CARA há nesta frase?
✅ Dois! Cara = man / tipo (pessoa) + caro = costoso. "Esse tipo é caro" — pode ser ambíguo!
6. Como se diz "Don Carlos" em português do Brasil?
✅ Seu Carlos — forma respeitosa e carinhosa para homem mais velho.
