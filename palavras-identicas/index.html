3 📐 Padrões — Aprende Um, Entende Mil

Clique para ouvir exemplos de cada padrão:

-ción / -sión → -ção / -são
información → informação · educación → educação · nación → nação
H inicial → F em PT
hablar → falar · hacer → fazer · habitar → habitar (alguns ficam iguais!)
-oche / -eche → -oite / -eite
noche → noite · leche → leite · hecho → feito · ocho → oito
LH sempre soa LI no Brasil
velho = véliu · olho = óliu · filho = fíliu · trabalho = trabáliu
