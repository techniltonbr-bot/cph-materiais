⚠️ Falsos Amigos — Atenção!

Propina ⚠️≠ gorjeta!Soborno (algo ilegal!) · Gorjeta = propina para mesero
Pagar ⚠️≠ cancelarPagar (≠ Cancelar em espanhol = cancelar aqui)
Borracha ⚠️Caucho / Borrador de lápiz · Bêbada = Borracha (em espanhol)
Embaraçada ⚠️Enredada · Grávida = Embarazada (em espanhol)
Esquisito ⚠️Raro / Extraño (≠ exquisito espanhol)
Polvo 🐙≠ póPulpo 🐙 · Pó = Polvo (en español)
Padre ⚠️Cura (iglesia) · Pai = Padre
Dívida ⚠️≠ dúvidaDeuda · Dúvida = Duda
Geral / General 🪖General (adjetivo) / General (ejército)
A gente / AgenteA gente = Nosotros · Agente = Agente · Gente = Todos
Trânsito / TráficoTrânsito = Tráfico vehicular · Tráfico = Tráfico ilícito
Venda / LendaVenda = Venta · Lenda = Leyenda
Preconceito ⚠️Prejuicio · Prejuízo = Perjuicio/Daños económicos
Piso / ChãoPiso = Baldosa · Chão = Piso (suelo)
Conserto / ConcertoConserto = Arreglo · Concerto = Concierto (orquesta)
Molestar ⚠️Acosar sexualmente (≠ molestar espanhol = incomodar)
Rapariga ⚠️Insulto no Brasil · Em Portugal = chica (não usar no Brasil!)
Pelé ≠ PelePelé (pe-LÉ) = jugador · Pele (pe-LI) = piel
MesmoMismo · "É mesmo?" = ¿En serio? ¿De verdad?
Direito / DiretoDireito = Derecho (ley/lado/correcto) · Direto = Directo
Ditado ⚠️Ditado = Dictado OU Dicho (depende contexto)
